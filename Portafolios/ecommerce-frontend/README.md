# E-commerce Frontend

## Descripción

Frontend de una tienda de e-commerce. Incluye:

- Página principal con listado de productos
- Tarjetas de productos con botón "Comprar"
- Página de detalle de producto
- Barra de navegación y footer

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Cómo probar

Abrir `index.html` en un navegador.
